## 构造函数
1. 构造函数是什么？（是什么）
   用于创建和初始化对象的特殊函数。
2. 为什么需要构造函数？（为什么）
   创建能使用共享属性和方法的对象。
3. 构造函数怎么使用？有那些特性？（怎么做）
   1. 创建：与普通函数区分首字母通常大写。
      ```js
      function Role() {
         this.name = "ycf";
         this.age = 28;
      }
      ```
   2. 使用：new配合函数调用，则会将函数作为构造函数调用。
      1. 创建初始化对象并绑定原型
      2. 默认this绑定初始化对象
      3. 默认返回this。
      ```js
      function Role(name, age) {
         this.name = name;
         this.age = age;
      }
      console.log(new Role("张三"，28))
      ```
   3. 特性：
      1. 原型：构造函数默认拥有的属性，用于存放共享的属性和方法。
      2. 原型链：对象访问属性和方法的规则，即自身没找到时会访问其构造函数原型，以此类推直至到头。
      3. `[[prototype]]`：对象含有的内置属性，它保存了对象构造函数原型的地址，实例访问原型就通过该属性。
      4. 原型链终点：JS引用数据都是Object演化来，因此Object构造函数的原型的原型null即是终点。
      5. 单独类型：类型检测时会将构造函数实例单独归类为该构造函数的类型。
