> 事件是组件编写交互逻辑的方式，通过向事件绑定函数实现执行逻辑代码

### React 事件

<CustomSandpack
  files={{
    "/App.js": app1,
  }}
/>

1. React 事件是 React 基于 DOM 事件自己实现的事件，事件处理函数通过 JSX 属性绑定事件
2. 事件处理函数写在组件最外层，绑定 React 事件时接收事件对象参数 event

### 自定义事件

<CustomSandpack
  files={{
    "/App.js": app2,
    "/Input.js": {
      code: input,
      hidden: false,
    },
  }}
  options={{
    showConsole: true,
  }}
/>

1. 基于 React 事件和 Props 实现，通过 JSX 属性绑定事件
2. 自定义事件可自定义参数，传递给事件处理函数
