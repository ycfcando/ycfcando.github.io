## 类
1. 类是什么？（是什么）
   更简洁直观创建和初始化对象的方式。
2. 为什么需要类？（为什么）
   相比于构造函数，类更加简洁直观且功能更加强大。
3. 类如何使用？有那些特性（怎么做）
   1. 创建：class关键字+首字母大写名称。
   2. 使用：new关键字+类的调用。
   3. 特性
      1. constructor：构造函数，类被调用时执行的函数，可接收调用时的传参。
      2. 公共方法：实例共享方法；类中直接添加方法即可。
      3. 实例属性：实例上的属性；类中直接添加属性即可。
      4. 静态属性/方法：类上的属性和方法；`static`关键字 + 属性/方法即可。
      5. 私有属性/方法：只能类中访问的属性和方法，`#` + 公共方法、实例属性、静态属性/方法即可。
      6. extends：类的继承，继承父类的公共方法和构造函数。
      7. super：继承的类在自定义构造函数时，要调用super来手动调用父构造函数，可以传参给父构造函数。
