## 数据类型
1. 数据是什么（是什么）？
   应对不同需求场景而构建的存储方式。
2. 数据类型有哪些？怎么使用？（怎么做）
   1. 类型：原始类型、引用类型。
      1. 原始类型：不能改变，直接存储在变量中。
         1. string：表示字符的数据。`const a = "你好"`
         2. number：表示数字的数据。`const a = 0`
         3. boolean：表示真假的数据。`const a = true || false`
         4. undefined：未赋值时的标识。`const a = undefined`
         5. null：对象为空时的标识。`const a = null`
         6. symbol：表示唯一值。`const a = Symbol("描述")`
         7. bigint：超过（±(2^53-1)）数字的数据。 `const a = BigInt("9007199254740991")`
      2. 引用类型：能改变，存储引用在变量中。
         1. Object：无序键值对集合。`const a = {"key": 0}`
         2. Array：有序值的集合。`const a = [1,2]`
         3. function：一段可执行的代码。
         注：引用类型都是Object的衍生。
   2. 自动装箱行为：基础数据用属性方法时，默认会用构造函数包装基础数据，使其能够使用属性和方法。
   3. 自定义数据类型：即自定义新的数据和规定其类型。
      1. 类型：JS在原型上规定了特定属性 `Symbol.toStringTag` 用于存放类型信息。
      2. 数据：类和构造函数可以自定义数据类型。
