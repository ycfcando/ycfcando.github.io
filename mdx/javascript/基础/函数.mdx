## 函数

1. 是什么？
   复用逻辑的方式。
2. 怎么做？
   1. 函数有两种表现形式：声明函数和箭头函数。
      ```js
      function greet(name) {
        return `Hello, ${name}!`;
      }
      const hello = () => {
        return `Hello, ${name}!`;
      };
      ```
   2. 以及两种保存形式：函数声明和函数表达式。
      ```js
      // 函数声明
      function greet(name) {
        return `Hello, ${name}!`;
      }
      // 函数表达式
      const hello1 = () => {
        return `Hello, ${name}!`;
      };
      const hello2 = function () {
        return `Hello, ${name}!`;
      };
      ```
      注：函数声明只能用声明函数。
   3. 函数可以接受参数，个数不限形式如下。
      ```js
      function greet(name, age) {
        return `${name}|${age}`;
      }
      greet("ycf", 28);
      ```
   4. 函数拥有内部参数 this 和 arguments。
      1. this：函数调用指向的对象，
         1. 声明函数默认是谁调用指向谁，但可以被 bind、call、apply 改变。
         2. 箭头函数是指向上一级上下文的 this。
         ```js
         function greet1() {
           console.log(this); // 没有调用默认是window对象调用，this指向window。
           const greet2 = () => {
             console.log(this); // 指向上一级上下文this，即window。
           };
         }
         greet1();
         ```
      2. arguments：数组类对象，按顺序存放接收的参数。
         ```js
         function greet1() {
           return `${arguments[0]}|${arguments[1]}`;
         }
         greet1("ycf", 28);
         ```
